# Module: EC2 & Bastion Host Cluster

Provides a customizable module to provision a group of EC2 instances in AWS. This module supports an optional Bastion Host for secure access, automated Security Group generation, and dynamic TLS private key creation for SSH access.

## Example Usage

### Basic Compute Cluster

```hcl
module "ec2_cluster" {
  source = "github.com/your-username/your-repo-name"

  group_name        = "web-app-cluster"
  default_ami_value = "ami-0123456789abcdef0" 

  instance_count    = 2
  instance_type     = "t3.micro"
  
  vpc_id            = "vpc-1234567890abcdef0"
  machine_net       = "subnet-0abcdef1234567890"
}
```

### Cluster with Bastion Host Enabled

```hcl
module "secure_ec2_cluster" {
  source = "github.com/your-username/your-repo-name"

  group_name        = "secure-app-cluster"
  default_ami_value = "" # Defaults to the latest Ubuntu 22.04 LTS AMI

  instance_count    = 3
  
  vpc_id            = "vpc-1234567890abcdef0"
  machine_net       = "subnet-0abcdef1234567890"

  bastion_host      = 1
  bastion_net       = "subnet-0987654321fedcba0"
  
  generated_new_ssh_key = 1
}
```

## Argument Reference

The following arguments are supported:

* `group_name` - (Required) The given name for this group of instances.
* `default_ami_value` - (Required) Pass any AMI ID if you want to customize at the AMI level. Leave as an empty string `""` to default to the latest Ubuntu 22.04 LTS image.
* `instance_count` - (Optional) The total number of ec2 instances used to create this load balancer. Default: `0`.
* `default_region` - (Optional) The region for these resources. Default: `"us-east-1"`.
* `vpc_id` - (Optional) ID of the VPC for this resource; otherwise it will fall to the default vpc. Default: `""`.
* `instance_type` - (Optional) The default type for instances if there is no argument passed. Default: `"t3.micro"`.
* `open_publicy_ssh` - (Optional) The option to open SSH public or not. Default: `0`.
* `bastion_host` - (Optional) Indicate that this machine has a bastion host or not. Default: `0`.
* `bastion_net` - (Optional) Network for bastion host - subnet id. Default: `""`.
* `machine_net` - (Optional) Network for main EC2 machine - subnet id. Default: `""`.
* `generated_new_ssh_key` - (Optional) Optional, choose to generate a new SSH key or not. Default: `1`.
* `pre_created_ssh_key` - (Optional) Specify whether to use a pre-created SSH key or not. Default: `0`.
* `default_public_ip_to_machine` - (Optional) Quick test, turn it on, otherwise, it is best to leave it as false. Default: `0`.
* `key_alogrithm` - (Optional) Algorithm used to generate SSH key pair. Must be `"RSA"`, `"ECDSA"`, or `"ED25519"`. Default: `"RSA"`.
* `key_bits` - (Optional) Number of bits used in the key generation process. Default: `4096`.

## Attribute Reference

In addition to all arguments above, the following attributes are exported:

* `instance_id` - A list containing the IDs of all created base EC2 instances.
* `instance_public_ip` - A list containing the public IP addresses of all created base EC2 instances.
